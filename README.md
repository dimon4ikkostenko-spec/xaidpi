# XAIDPI

[![Release](https://img.shields.io/badge/release-v1.00_stable-blue)](https://github.com/KostenkoDev/XAIDPI/releases)
[![Platform](https://img.shields.io/badge/platform-Windows_10%2F11-lightgrey)]()

**XAIDPI** is a lightweight, modern network optimization tool designed to improve connection stability and fix packet loss issues for various high-latency services. Built with a clean Apple-style interface, it operates efficiently in the background.

---

## üá∫üá∏ English

### Features
* **Modern UI:** Clean, minimalist interface inspired by Apple design guidelines using `customtkinter`.
* **One-Click Operation:** Simple Start/Stop mechanism with automated driver management.
* **Stealth Mode:** Operates silently with minimal system resource usage.
* **Multi-Strategy Support:** Includes various presets for different network conditions.
* **Theme Support:** Switch between Dark, Light, System, or Chameleon (Multicolor) modes.

### Installation & Usage
1. Download the latest `XAIDPI.exe` from the [Releases](https://github.com/KostenkoDev/XAIDPI/releases) section.
2. Run the executable as Administrator.
3. Select a strategy from the dropdown menu (e.g., `SIMPLE FAKE ALT2`).
4. Click **START ENGINE**.

### Developer Build
To build from source:
1. Install Python 3.10+.
2. Install dependencies: `pip install customtkinter darkdetect auto-py-to-exe`.
3. Rename your engine binary to `engine.exe` and place it in `bin/`.
4. Run `main.py`.
